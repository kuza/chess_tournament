Chess tournament был создан как решение тестового задания для wargaming.net.

Chess tournament это приложение по проведению турнира по шахматам по швейцарской системе.

Основные возможности приложения:

  • Два интерфейса: для судей с возможностью управления и турниром и для простых пользователь с возможностью просмотра информации по турнирам.
  • Автоматический расчет пар при создании судьёй нового раунда в турнире с учетом цвета (не более двух одинаковых цветов подряд).
  • Расчет изменения рейтинга Эло после каждой партии.
  • Турнирная таблица с указанием текущего места, количества заработанных очков и нового рейтинга.

Применяемый к-фактор для расчета рейтинга Эло:

  • 32 если текущий рейтинг меньше 2100.
  • 24 если текущий рейтинг больше 2100 но меньше 2400.
  • 16 если текущий рейтинг больше 2400.

Разработчик: k.v-v@tut.by

Chess tournament was created as a solution to the test task for wargaming.net.

Chess tournament is an application for the chess tournament by Swiss-system.

General features:

  • Two interface: for referees (they can manage tournament) and for simple users (they have readonly permissions).
  • Automatic calculation of pairs when referee creates a new round in the tournament. Player gets color is different from the previous game. If this impossible player gets same color but not more than two consecutive.
  • Elo rating calculated after each game.
  • Tournament table has current place, current score and current Elo rating.

K-factor for Elo rating:

  • 32 if current Elo rating less than 2100.
  • 24 if current Elo rating more than 2100 but less that 2400.
  • 16 if current Elo rating more than 2400.

Developer: k.v-v@tut.by
